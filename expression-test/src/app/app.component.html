MindSung Expression Test

Create an expression (no input) using:
$f(<param_value_or_expr>, ...)
or with scope variables:
$withVars({ name: <var_name>, value: <value_or_expr> }, ...).$f(<param_value_or_expr>, ...)

Create a transform expression (with input) using:
$fx(<input_expr>, <param_value_or_expr>, ...)
or or with scope variables:
$withVars({ name: <var_name>, value: <value_or_expr> }, ...).$fx(<input_expr>, <param_value_or_expr>, ...)

An object containing sample data and transforms can be accessed as $sample
